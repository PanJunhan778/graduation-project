### M1 — 基础搭建与登录功能 ✅

**完成日期**：2026-04-09

**前端交付物**：
- 项目脚手架：Vite + Vue 3 + TypeScript + Element Plus + Pinia + Vue Router
- 全局样式：按 DESIGN.md 规范覆盖 Element Plus CSS 变量，Inter 字体引入
- Axios 封装：Token 自动注入、401 拦截跳转、统一错误提示
- 路由守卫：未登录重定向至 /login，已登录跳过登录页
- 主布局骨架：MainLayout（可折叠 Sidebar + Topbar + RouterView），按角色动态渲染菜单
- 登录页：角色视觉引导选择器（管理员/老板/员工）+ 表单校验
- 注册页：Staff 专属，含 6 位企业码、密码复杂度校验

**后端交付物**：
- 项目配置：application.yml（MySQL、MyBatis-Plus、Sa-Token JWT、CORS）
- 数据库：7 张表完整 DDL（schema.sql）+ admin 预置账号
- 7 个实体类：Lombok + MyBatis-Plus 注解，金额 BigDecimal，逻辑删除
- 统一响应体 Result\<T\>，全局异常处理器
- Sa-Token JWT Mixin 模式，路由拦截（排除 auth 路径）
- MyBatis-Plus 多租户拦截器 + 分页插件 + 自动填充
- AuthController：POST /api/auth/login + POST /api/auth/register
- BCrypt 密码加密，Caffeine 登录失败锁定（5 次 / 15 分钟）

**调试修复记录**：
- CORS 配置：`setAllowedOrigins` 改为 `setAllowedOriginPatterns("http://localhost:*")`，适配 Vite 动态端口
- JDBC 编码：`characterEncoding=utf8mb4` 改为 `characterEncoding=UTF-8`（Java Charset 不识别 utf8mb4）
- Sa-Token 拦截器：`SaRouter.notMatch()` 改为 Spring MVC 原生 `excludePathPatterns()`
- Admin 密码哈希：使用 Sa-Token BCrypt 工具重新生成 `Admin@123` 对应的正确哈希

  - 底部分页器：支持 20/50/100 条切换
- 请求层增强（request.ts）：`responseType: 'blob'` 时跳过 JSON 业务码解析，直接返回原始响应；业务失败且响应体含 `data` 时不自动 ElMessage，由导入页自行展示错误明细

**后端交付物**：
- Mapper：FinanceRecordMapper（extends BaseMapper\<FinanceRecord\>，无 XML）
- DTO：FinanceCreateDTO（type / amount / category / project / date / remark，JSR 校验，金额 @DecimalMin("0.01")）；BatchDeleteDTO（ids 非空列表）
- VO：FinanceRecordVO（含 date、createdTime 的 @JsonFormat）
- Service：FinanceService 接口 + FinanceServiceImpl 实现
  - listRecords：分页 + type / category / startDate / endDate 筛选，按日期与 id 倒序；租户由 MyBatis-Plus 自动拼接 company_id
  - createRecord / updateRecord：校验 income/expense；update 时校验记录存在
  - deleteRecord / batchDelete：逻辑删除（MyBatis-Plus）
  - importExcel：Apache POI 解析“导入数据”Sheet，按表头取值，支持月度/季度/年度所属期、条件校验缴纳日期、文件内重复校验、数据库重复校验；任一行失败则整批拒绝导入并返回 `{row, error}` 列表
  - downloadTemplate：Apache POI 生成富说明模板，包含字段说明、必填/选填标识、示例、下拉约束、批注和列宽冻结配置
  - importExcel：Hutool ExcelReader 解析，逐行强校验；任一行失败则整批不入库，返回 code=400 与 `[{row, error}]`；支持中文「收入/支出」与 income/expense；日期兼容字符串与 Excel 日期单元格
  - downloadTemplate：Apache POI 生成双 Sheet 富说明模板，包含字段说明、必填/选填标识、示例、下拉约束、批注和列宽冻结配置
- Controller：FinanceController（类上无 @SaCheckRole，Owner/Staff 登录即可）
  - GET /api/finance/list — 分页财务列表
  - POST /api/finance — 单笔新增
  - PUT /api/finance/{id} — 单笔编辑
  - DELETE /api/finance/{id} — 单笔删除
  - POST /api/finance/batch-delete — 批量逻辑删除
  - POST /api/finance/import — Multipart 导入
  - GET /api/finance/template — 下载 .xlsx 模板
